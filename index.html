This page introduces my Android project, Map Timeline Tool. It is an offline-first point logging app with map view, list management, CSV and ZIP import/export, and local data storage.

Project Highlights

Offline First

Point data stays on-device. Tile caching and area downloads keep core map usage available in weak networks.

Portable Data

Export and import CSV, GeoJSON, KML, KMZ, and ZIP formats for easy collaboration with other tools.

Fast Capture

Quick-add notification, photo attachments, and tag management make field collection much smoother.

All Functions

  • Display marks on either normal map or satellite map.
  • Take photos or record phone sensor data when adding points.
  • Download selected map areas so they can be displayed offline.
  • Create tags to sort points.
  • Backup, restore, and share full records.
  • Optimized for quick adding.

Advantages

  • Available on Android devices.
  • Very lightweight and really fast.
  • All features are available offline (except map loading).
  • MIT open-source, free and safe, everyone can use the software.

Tech Stack

  • Android
  • Kotlin
  • Jetpack Compose
  • Room
  • osmdroid
  • OpenStreetMap

Project Snapshot

Map Timeline Tool is an offline-first Android app for manual point logging. It captures coordinates with notes, photos, and tags.
