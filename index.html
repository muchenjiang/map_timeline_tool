Hi, I'm Muchen Jiang

This page introduces my Android project, Map Timeline Tool. It is an offline-first point logging app with map view, list management, CSV and ZIP import/export, and local data storage.

View Source Highlights

Project Highlights

Offline First

Point data stays on-device. Tile caching and area downloads keep core map usage available in weak networks.

Portable Data

Export and import CSV, GeoJSON, KML, KMZ, and ZIP formats for easy collaboration with other tools.

Fast Capture

Quick-add notification, photo attachments, and tag management make field collection much smoother.

Project Snapshot
