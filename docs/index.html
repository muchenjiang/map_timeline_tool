你好，我是 Muchen Jiang

这个页面用于介绍我的 Android 项目 Map Timeline Tool。 它是一款离线优先的打点记录工具，支持地图查看、列表管理、CSV/ZIP 导入导出，以及本地数据保存。

查看源码 项目亮点

项目亮点

离线优先

点位数据保存在设备本地。地图瓦片支持缓存与下载，弱网环境仍可使用核心能力。

数据可迁移

支持 CSV、GeoJSON、KML、KMZ、ZIP 导入导出，方便与其他地图工具协同。

快速记录

支持通知栏快速打点、照片关联与标签管理，适合现场采集和路径回溯。

English Snapshot

Map Timeline Tool is an offline-first Android app for manual point logging. It helps capture coordinates with notes, photos, and tags, then export data for further analysis.
